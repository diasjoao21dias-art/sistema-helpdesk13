==================================================================
         GUIA DE INSTALAÇÃO - SISTEMA OLIVION
         Gestão de Chamados/Tickets com Sistema de Licenças
==================================================================

📋 PRÉ-REQUISITOS
- Windows 10/11 ou Linux
- Python 3.8+ instalado (https://python.org/downloads)
- Conexão com internet (para instalar dependências)

🔧 PASSO A PASSO DE INSTALAÇÃO

1️⃣ BAIXAR O SISTEMA
   - Copie toda a pasta do sistema para seu computador
   - Exemplo: C:\Sistema-Olivion\

2️⃣ INSTALAR DEPENDÊNCIAS
   Abra o terminal/prompt na pasta do sistema:
   
   Windows:
   cd C:\Sistema-Olivion\Sistema-OS-v9\TesteProgram\
   pip install -r requirements.txt
   pip install pyopenssl
   
   Linux:
   cd ~/Sistema-Olivion/Sistema-OS-v9/TesteProgram/
   pip3 install -r requirements.txt
   pip3 install pyopenssl

3️⃣ LIMPAR BANCO PARA SUA MÁQUINA (IMPORTANTE!)
   Execute APENAS na primeira instalação:
   
   Windows:
   python LIMPAR_BANCO.py
   
   Linux:
   python3 LIMPAR_BANCO.py

4️⃣ EXECUTAR O SISTEMA
   Windows (na pasta TesteProgram):
   python USAR.py
   
   Linux (na pasta TesteProgram):
   python3 USAR.py

🌐 ACESSAR O SISTEMA
   Abra seu navegador e vá para:
   http://localhost:3003

🔑 LOGIN PADRÃO
   Usuário: admin
   Senha: admin
   
   ⚠️  IMPORTANTE: Altere a senha após o primeiro login!

🔒 DOWNLOADS SEGUROS (PDF/Excel)
   Para evitar avisos de "download inseguro" no navegador local:
   
   Windows:
   set ENABLE_LOCAL_SSL=1
   python USAR.py
   
   Linux:
   export ENABLE_LOCAL_SSL=1
   python3 USAR.py
   
   - Primeira vez: Aceite o certificado autoassinado no navegador
   - Acesse: https://localhost:5000 (com HTTPS)
   - Downloads de relatórios funcionarão sem avisos de segurança
   
   💡 Se não ativar SSL, use: http://localhost:5000 (downloads podem ter aviso)

⚙️  CONFIGURAÇÕES AVANÇADAS

📁 ESTRUTURA DE PASTAS
   - static/uploads/ → Imagens dos chamados
   - static/relatorios/ → Relatórios gerados
   - templates/ → Interfaces do usuário
   - sistema_os.db → Banco de dados SQLite

🔐 PRIMEIRO ACESSO
   1. Acesse http://localhost:3003
   2. Login: admin / admin
   3. Vá em Admin > Usuários
   4. Altere a senha do admin
   5. Crie operadores e usuários

🔑 CONFIGURAÇÃO DE SEGURANÇA (OBRIGATÓRIO)
   Para uso em produção, altere a chave secreta:
   1. Crie arquivo .env na pasta TesteProgram
   2. Adicione: FLASK_SECRET_KEY=sua-chave-secreta-forte-aqui
   3. Ou edite app.py linha 94: app.secret_key = "sua-chave-forte"

👥 CRIAR USUÁRIOS
   - Admin > Usuários > Novo Usuário
   - Tipos: admin, operador, usuario
   - Atribua setores aos operadores

📋 GERENCIAR SETORES
   - T.I → Problemas técnicos
   - Manutenção → Manutenção predial
   - CCIH / SESMT → Segurança do trabalho
   - Telefonia → Telecomunicações

==================================================================
🔧 SOLUÇÃO DE PROBLEMAS
==================================================================

❌ ERRO: "Porta 3003 em uso"
   → Feche outros programas usando a porta ou altere em USAR.py

❌ ERRO: "ModuleNotFoundError"
   → Execute: pip install -r requirements.txt

❌ ERRO: "Permission denied"
   → Execute como administrador (Windows) ou com sudo (Linux)

❌ ERRO: "Database locked"
   → Feche completamente o sistema e inicie novamente

❌ Sistema lento
   → Feche navegador e abra novamente
   → Reinicie o sistema com python USAR.py
